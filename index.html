Project VSIM - Vision Simulator

VSIM - Vision Simulator a simulation app for several vision disorders. Application will help the patients, their parents, relatives and doctor understand these conditions using the live video from the camera of the device. The app will allow the user to not only choose the disease type but also control the disease severity using a slider. Application was developed using Python 3.5.2, OpenCV 3.4.0 and Kivy 1.10.0.

Following simulations were carried out:

  • Cataract Vision.
  • Diabetic Retinopathy Vision.
  • Glaucoma Vision.
  • Age-Related Macular Degeneration.

Project HOPE

Project HOPE is intended for People with Speaking Disability, allowing them to express their feelings through computer aided voice assistant. Application scans through different group of letters, a person has to click the mouse anywhere on the application to select the corresponding group and it jumps from level to level, back-and-forth, until the person forms the sentence to speak at final stage. Frontend was developed using Tkinter library interface for Python 3.

Aim of the project was to make a desktop friendly application using python, which has the following frontend needs :

  • Level 1, 2, 3 and 4 design.
  • Scanning through each level.
  • Getting text from level 1 to level 2 to level 3 to level 4 and erasing back.
  • Start and Stop scan functionalities.
  • Avoiding crash of application due to several errors.
  • Successful deployment on any OS environment.
