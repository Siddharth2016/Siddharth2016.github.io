Project HOPE

Project HOPE is intended for People with Speaking Disability, allowing them to express their feelings through computer aided voice assistant. Application scans through different group of letters, a person has to click the mouse anywhere on the application to select the corresponding group and it jumps to level 2 of execution. Now the person is asked to select a particular letter from the group (achieved through scanning) and after selection of letter, backend function to predict the word is called (Backend developed by another team), here level 3 type execution takes place. After proper forming of the sentence (if user wants to select different word he/she can go back to level 1 and proceed in the same order) level 4 is called to execute text-to-speech module. Frontend was developed using Tkinter library interface for Python 3.

Aim of the project was to make a desktop friendly application using python, that has the following frontend needs :

  • Level 1, 2, 3 and 4 design.
  • Scanning through each level.
  • Getting text from level 1 to level 2 and erasing back.
  • Start and Stop scan functionalities.
  • Avoiding crash of application due to several errors.
  • Successful deployment on any OS environment.
